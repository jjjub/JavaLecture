표준 API 함수형 인터페이스

1. Consumer
- (매개변수) -> {}
2. Supplier
- () -> { return 값; }
3. Function
- (매개변수) -> { return 값; }
4. Operator
- (매개변수) -> { return 값; }

- 매개변수와 반환값이 동일한 타입
5. Predicate
- (매개변수) -> { return 값; }

- 반환값이 boolean
